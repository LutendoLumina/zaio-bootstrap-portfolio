# Clone Your Future – Bootstrap Portfolio

This project is a personal developer portfolio built using **HTML, CSS, and Bootstrap**.  
The purpose of the project is to create a clean and responsive website that showcases my skills, projects, and contact information.

The portfolio acts as a **professional profile page** that can be shared with employers, recruiters, and collaborators.

---

## Project Overview

The portfolio contains several sections designed to highlight a developer’s profile:

- A **Hero section** introducing the developer
- An **About Me section** with personal information and skills
- A **Projects grid** displaying completed work
- A **Contact section** with social links and a form
- A **Footer** for attribution

The website is fully responsive and adapts across different screen sizes ranging from **mobile devices to large desktop screens**.

---

## Features

- Responsive layout built with **Bootstrap**
- Sticky navigation bar
- Hero section with full-width background image and overlay
- About Me card with profile information and skills badges
- Projects grid displaying multiple projects
- Buttons linking to live websites and GitHub repositories
- Contact section with social media icons
- Contact form with HTML validation
- Dark mode support using `prefers-color-scheme`

---

## Technologies Used

- **HTML5**
- **CSS3**
- **Bootstrap 5**
- **Bootstrap Icons**

## Project Sections

### Hero Section
A full-width hero section introducing the developer with a background image, dark overlay, and centered text.

### About Me
A two-column card layout containing a profile image, personal details, skills badges, and a short introduction.

### Projects
A responsive grid layout displaying six projects with images, descriptions, and links to both live websites and source code.

### Contact
A section that includes social media links and a contact form for users to send messages.

### Footer
A simple footer with attribution text.

---

## Responsive Design

The layout adjusts based on screen size:

- **Large screens (lg)** → 3 project cards per row
- **Medium screens (md)** → 2 project cards per row
- **Small screens (sm)** → 1 project card per row

## How to Run the Project

1. Clone the repository:
git clone https://github.com/LutendoLumina/lutendo-portfolio.git


2. Open the project folder.

3. Open `index.html` in your browser.

No additional setup is required since this is a **static website**.
